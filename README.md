# AI Vision Classifier Pro

A professional deep learning web application built with Python, TensorFlow, and Streamlit.  
The app allows users to upload an image and receive a classification prediction with confidence scores and probability distribution.

## Features

- Upload an image and classify it in real time
- Displays top prediction and confidence score
- Shows top 3 predicted classes
- Probability chart for all supported classes
- Clean professional UI for deployment

## Tech Stack

- Python
- TensorFlow / Keras
- Streamlit
- NumPy
- Pillow
